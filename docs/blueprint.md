# **App Name**: QuickLinker

## Core Features:

- Short URL Creation: Allows users to input a long URL, generate a unique short code, and optionally specify a custom alias and an expiration date. Displays the generated short URL.
- Short URL Redirection: Facilitates high-speed redirection from a short URL to its original long URL, prioritizing low latency and efficiency.
- Click Analytics Dashboard: Provides a visual dashboard to display key metrics for each short URL, including total clicks, daily click breakdown, and anonymized user details like referrer or device type.
- Custom Alias Moderation: Leverages an AI tool to automatically detect and flag custom aliases that may be offensive, trademarked, or on a list of reserved keywords during the creation process.
- Alias Conflict Resolution: Handles cases where a user attempts to create a custom alias that is already in use, providing appropriate feedback.
- Optional Expiration Dates: Enables users to set an expiration date for their short URLs, after which the link will become inactive and return a 404.

## Style Guidelines:

- The visual theme is dark, suggesting professionalism, technological focus, and efficiency. The background color, #161B1F, is a very dark, subtle blue-grey, providing a deep canvas.
- The primary accent color, #1F5C99, is a rich, deep blue, conveying stability, trustworthiness, and a high-tech aesthetic, used for key interactive elements and branding.
- A vibrant cyan, #42E1FF, serves as an accent color, bringing a dynamic and modern energy to calls-to-action, alerts, and data highlights, contrasting effectively with the darker palette.
- Headlines and body text will use 'Inter', a modern grotesque sans-serif, chosen for its excellent readability across all sizes and its neutral, objective aesthetic, aligning with the app's focus on precision and data.
- Icons should be minimalist, clean, and geometric, using line art where appropriate to maintain a modern and efficient feel that complements the data-centric nature of the application.
- The layout emphasizes clarity and hierarchy. A clean, uncluttered design uses ample negative space to make information digestible, with data-rich sections organized into intuitive cards or tables for easy scanning and comparison.
- Subtle, fast, and smooth micro-interactions should provide instant feedback without distraction. Animations will be minimal, designed to enhance the sense of speed and responsiveness of the application.
